En un archivo de texto separado que debes crear, escribe explicaciones de los siguientes conceptos como si se lo estuvieras explicando a un niño de 12 años. Hacer esto te ayudará a descubrir rápidamente cualquier agujero en tu comprensión.

Variables: Cualquier cosa u objeto puede tener diferentes valores, por eso decimos que son variables. Por ejemplo, una manzana puede valer 5 pesos en un mercado y 7 pesos en otro. En las matemáticas y la computación es más fácil manejar variables en lugar de adoptar un valor fijo; ya que sabemos que este valor va a cambiar en el tiempo. 

Strings: Es la forma que en la informática se refieren a las "palabras". Digo "palabras" entre comillas porque no necesariamente tienen que tener un significado en algún idioma en especifico. Por ejemplo un string pueden ser palabras como "auto" o "arbol" pero tambien podrían ser algo como "jukasid" o "xyz"; tambien podrían ser el nombre de una variable o una frase completa, todo depende de lo que desee el programador.

Funciones (argumentos, return): Cuando se está programando, primero se declaran unos valores, ya sean constantes o variables. Luego, con estos valores el computador puede comenzar a "jugar" con ellos para darnos un nuevo resultado. Para eso se necesita una "función", que toma estos valores y los transforma. A las instrucciones para transformar estos valores se llaman los "argumentos de la función". Por ejemplo, teniendo dos valores diferentes "a" y "b", un argumento podría ser sumarlos; o podría también ser que otro argumento sea los multiplique. La función que usemos será la que determinará como se usan esos valores, para luego tener un resultado, que será reportado por medio de un argumento llamado "return".

Declaraciones if: Al usar estas funciones, nos daremos cuenta que muchas veces tenemos que comparar un resultado con un valor para saber cómo proceder. Es una forma de que de antemano el computador "decida" que camino escoger. "If" quiere decir Si en inglés, pero es un si con condiciones (no es el si de afirmar positivamente). Por ejemplo, podemos hacer una funcion que nos suma y nos promedia las notas de los estudiantes. SI el resultado es mayor o igual a 3, la función hace un "return" APROBADO. O si el resultado es menor de 3, el "return" será REPROBADO. 

Valores booleanos (true, false): Del mismo modo que las declaraciones IF, se pueden usar otros tipos de valores llamados booleanos que son o verdaderos "true" o falsos "false", con lo que podemos aumentar el poder de toma de decisiones de nuestro programa. Siguiendo con el ejemplo anterior, imaginemos que la función que nos promedia las notas se usa para varias materias y el return "APROBADO" es igual a un TRUE. Si todas las materias salen en APROBADO, se hace un return de "PASA AL SIGUIENTE CURSO". En cambio, si la función pasa algún return "REPROBADO" que es igual a un FALSE, se hace un return "EN SUSPENSO". Acá se puede ver cómo a partir de un programa sencillo se puede manejar una cantidad grande de información (digamos 50 alumnos cursando 9 materias, cada una con 6 notas de curso) hasta procesarla para tener un resultado corto y objetivo.
